# dot-flux
My f.lux configuration

Feel free to use it as a starting point for your own f.lux configuration!

## Install
**Requirements:** [Ellipsis][ellipsis] `(>=v1.9.0)`

``` shell
$ ellipsis install groggemans/dot-flux
```
